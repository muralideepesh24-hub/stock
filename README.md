The Stock Dashboard is a web-based application developed to provide an interactive and visually appealing way to monitor and analyze stock market data. It is designed to simplify the understanding of stock price movements by presenting information through charts and structured statistics instead of raw numerical data. The application allows users to search for stock symbols such as AAPL or TSLA and instantly view their price trends, percentage changes, and trading volume. It also includes a watchlist feature that enables quick access to popular stocks like Apple, Tesla, Amazon, and Microsoft. The dashboard uses a line chart to represent stock price variations over time, helping users easily identify patterns and trends. One of the key features of this project is the simulation of real-time updates, where stock values refresh automatically at regular intervals, giving users an experience similar to real financial platforms. The project is built using HTML for structuring the webpage, CSS for designing a modern dark-themed interface, and JavaScript for handling dynamic functionality such as generating data, updating values, and rendering charts using the Chart.js library. The system is user-friendly and does not require any additional software installation, as it runs directly in a web browser. Overall, the Stock Dashboard serves as an educational tool that demonstrates how financial dashboards work and how web technologies can be used to create interactive data visualization applications, making it especially useful for students and beginners learning full-stack development
